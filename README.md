## Overview  
macOS integrates multiple **hidden AI-driven subsystems**,  
including on-device intelligence engines, continuous ML background analysis,  
context recognition, semantic indexing, object detection and automated automation tools.

These modules operate quietly and are not shown as apps or settings.

Full system breakdown is available in the guide above.

---

## Core AI Functions  

### • Live Text AI Recognition  
Extracts text from images, screenshots, UI surfaces and camera input.

### • Visual Lookup Intelligence  
Identifies objects, plants, animals, landmarks, categories and scenes.

### • Spotlight ML Ranking System  
Learns user behaviour and ranks files, apps and actions based on context.

### • Siri Suggestion Engine  
Generates predictive actions, shortcuts, location awareness, routine detection.

### • On-Device Speech Neural Engine  
Processes speech-to-text and command understanding fully offline.

### • System Context Awareness  
Understands patterns, habits, app usage, calendars, notifications and time-based routines.

## Additional Hidden Modules  

### • Image Embedding Model  
Creates vector embeddings for image similarity and recognition tasks.

### • Autofill Prediction AI  
Suggests information, contacts and form entries contextually.

### • Background Content Scanner  
Analyzes photos, messages, files and screenshots for semantic data.

Organizes files based on content, type and usage patterns.
